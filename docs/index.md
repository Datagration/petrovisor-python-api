# Introduction to petrovisor

The petrovisor package provides Python bindings for the PetroVisor platform API.

PetroVisor is a platform for data integration, visualization, and analysis in the oil and gas industry.

## Features

- Connect to PetroVisor platform instances
- Work with time series data, petroleum engineering datasets, including reservoir models
- Train and run machine learning models
- Build and run automated workflows
